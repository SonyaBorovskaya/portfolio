Учебный проект на тему корпоративного обучения персонала организации.
Функционал: 
- регистрация и авторизация пользователей; 
- создание и размещение статей; 
- создание отчетов с помощью SQL-запросов; 
- прохождение тестирования по темам с фиксированием результатов. 
При разработке использовались фреймворки: SematnicUI, jQuery. Для написания стилей использовался SCSS
Сайт связан с бд, дамп которой представлен в файле dumb_db.sql.
