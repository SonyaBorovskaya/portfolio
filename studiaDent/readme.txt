Учебный проект на тему взаимодействия стоматологической клиники с клиентами.
Функционал: 
- регистрация и авторизация пользователей; 
- автоматическое подставление данных об авторизированном пользователе в форму заявки; 
- создание отзывов и из размещение; 
- административная панель с возможностью добавления, удаления и редактирования услуг; просмотром заявок. 
При разработке использовались фреймворки: SematnicUI, jQuery. Для написания стилей использовался SCSS
Сайт связан с бд, дамп которой представлен в файле dumb_db.sql.
